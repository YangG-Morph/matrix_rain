# Modes
- Wallpaper - press **W**:
  * **Only works on Windows**
  * ***Experimental** - will need to close with task manager*
  * or run through command line for easy exiting via CTRL+C


- Fullscreen/Windowed - press **F**:
  * Press F to restore
  * Press Q to quit
  * Press D to display FPS
